# Find the Best Route for Police's Patrol
## 功能描述
本計畫發現警察巡邏方式有一種為不定線巡邏，這種巡邏方式沒有固定路線，這有個缺點是效率不高，因此希望有個系統能根據犯罪相關地點數據來規劃一條最佳巡邏路線進而幫助警察可以最有效率的執勤。
